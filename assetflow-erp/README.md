# AssetFlow ERP Frontend

This folder contains the React + TypeScript + Vite frontend for AssetFlow ERP.

For full features, installation steps for Linux, Windows, and macOS, environment variables, seed users, and run commands, see the root project README:

```text
../README.md
```

Common local commands:

```bash
npm install
cp .env.example .env
npm run dev
npm run build
npm run lint
```

For local development, set `.env` to:

```env
VITE_API_URL=http://localhost:3001/api
```
